# 💇 美容室予約システム（Backend）

本リポジトリは、**特定の美容室を想定した予約管理システムのバックエンド**です。  
ユーザーがメニュー・デザイナー・日時を選択し、  
予約の重複やデザイナーの休日を考慮しながら予約できる仕組みを実装しています。

---

## 📌 概要（Overview）

- 美容室の予約業務を想定した Webアプリケーション
- 顧客・デザイナーなど **役割別の機能分離**
- 実運用を意識した **予約重複チェック・時間計算ロジック** を実装

---

## 🛠 使用技術（Tech Stack）

### Backend
- PHP  
- MySQL  

### Infrastructure / Environment
- Docker  
- Nginx  
- PHP-FPM  

### Tools
- Git / GitHub  
- VS Code  

---

## 👤 ユーザー区分（Role）

- **Client（顧客）**  
  - メニュー・デザイナー・日時を選択して予約作成
  - 自分の予約履歴を確認

- **Designer（デザイナー）**  
  - 自分に紐づく予約の確認
  - 休日（対応不可時間）の管理

---

## ✨ 主な機能（Features）

- 予約作成機能  
- 予約重複チェック（同一時間帯・デザイナー）
- デザイナーの休日を考慮した予約制御
- サービス所要時間を考慮した終了時刻の自動計算
- 予約一覧表示（過去 / 未来）
- 役割別（Client / Designer）表示制御

---

## 🧠 工夫したポイント（Key Points）

- **DB設計を意識したテーブル分離**  
  （Reservation / Service / User など）
- サービスの所要時間を合算し、  
  **予約終了時刻を自動計算するロジック**を実装
- 実際の美容室運用を想定し、  
  「予約できないケース」を事前に判定
